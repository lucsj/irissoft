Télécharger les sources des programmes

Comme pour tout logiciel libre, il est possible de consulter les sources de chacuns de nos programmes, pour cela il y a deux solution :

Le dépôt web contenant les sources

Nous avons mis à vôtre disposition un dépôt web vous permettant de naviguer dans les sources, et de les télécharger, cependant elles sont mise à jour que de temps en temps, voici le lien pour s'y rendre : Naviguer sur les sources d'IrisSoft

Dépôt IrisSoft sur GitHub

Pour télécharger nos sources à partir de github il vous faut installer le logiciel git, en suite rendez-vous dans le dossier où vous voulez mettre les sources puis ouvrez le menu contextuel et choisissez Git Bash Here, ce qui veut dire Git Bash ici, ce qui ouvrera une console à partir du dossier que vous avez précédament sélectionné

En suite entrez les deux lignes de commandes suivantes :

git init

git clone https://github.com/lucsj/irissoft.git

Pour mettre à jour les sources vous entrerez ceci :

git pull origin master

C'est tout pour l'utilisation de git affin d'avoir les sources des dernières versions de nos programmes, mis à jour en direct pendant le développement de ces derniers.
